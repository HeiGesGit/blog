# 从零搭建一个博客

## 1. 使用的技术

1. Vue全家桶

- 使用vuex状态管理,
  - vue-router进行路由的跳转
  - axios请求数据

- Element-UI
  - 大多数组件使用的element-ui
- webpack
  - 打包工具,使用它进行一些配置
- Nodejs
- Express
  - express搭建的后台框架工具
- Mongodb
  - 对数据的增删查改
